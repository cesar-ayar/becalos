Security

The Enhance control panel has built in tools to help harden server security and prevent vulnerabilities in one website affecting other sites, even within the same customer subscription.

  1. ✔️ Server role and website containerisation
  2. ✔️ Brute force protection
  3. ✔️ ModSecurity with default OWASP ruleset
  4. ✔️ Automatic Let’s Encrypt SSL certificate provisioning
  5. ✔️ Role based user access
